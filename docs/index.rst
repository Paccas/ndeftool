===================================
NDEF Creation and Modification Tool
===================================

.. _ISCL: http://choosealicense.com/licenses/isc/
.. _GitHub: https://github.com/nfcpy/ndeftool
.. _PyPI: https://pypi.python.org/pypi/ndeftool

The **ndeftool** is a command line utility for creating, modifying and printing
NFC Data Exchange Format (NDEF) Records. It is licensed under the `ISCL`_,
hosted on `GitHub`_ and installable from `PyPI`_.

Installation
------------

.. code::

   pip install -U ndeftool

Usage
-----

.. code::

   Usage: ndeftool [OPTIONS] COMMAND1 [ARGS]... [COMMAND2 [ARGS]...]...
